This is the repository for the simulation codes for the course on Adaptive Optics at the University of Milano-Bicocca. All are jupyter notebooks that can be uploaded on the Colab framework at  https://colab.research.google.com/
List of files with aims.
1.	ChromaticAberr.ipynb. It is made to use the Sellmeier approximation for the refractive index. It computes RI for BK7 and SF2 glasses. It should be complemented with the computation for the eye lens, for which formula for a thick lens are needed.
2.	FourierPrimer1.ipynb. Simple example of the use of the 2D Fourier Transform on a sample image.
